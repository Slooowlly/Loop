Visual Companion

Outras categorias no mesmo calendário.

Estas variações mantêm a sua categoria como protagonista e mostram as corridas das demais categorias com menos peso visual. O objetivo aqui é sentir o limite de densidade quando entram todas as categorias.

Densidade
Opção A

Pontos fantasmas

Cada dia com outras corridas recebe só pequenos pontos semitransparentes. É a leitura mais discreta e menos invasiva.

Opção B

Mini etiquetas apagadas

As outras categorias aparecem como tags mínimas com opacidade baixa. Fica mais informativo, mas já começa a chamar atenção.

Opção C

Névoa de contexto

Não tenta listar tudo. Só sugere “há muito movimento aqui” com uma aura sutil e uma contagem, deixando a célula limpa.
